# 🚖 GoodCabs Taxi Performance & Analytics

## 📌 Project Overview
This Power BI dashboard project analyzes taxi operations, passenger behavior, revenue trends, customer ratings, and city-wise performance for GoodCabs.

The dashboard provides actionable business insights to improve operational efficiency and passenger experience.

---

# 📌 Key Insights
- Jaipur generated the highest revenue and trip count.
- Repeat passengers contributed significantly to overall trips.
- Weekend trips were slightly higher than weekday trips.
- Mysore and Jaipur showed strong passenger ratings.
- Some cities showed lower retention rates requiring business improvements.
